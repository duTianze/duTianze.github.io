用户理所当然地认为计算机同时做多件事。 当其他应用程序在下载文件和播放音乐时，他们认为可以继续在文字处理器中工作。
即使是单个应用程序，通常也被期望可以同时做多件事情。例如，播放音乐的应用程序必须同时从网络中读取数字音频、解压、播放和显示。
即使是文字处理程序也应该随时准备响应键盘和鼠标事件，无论它在重新格式化文本或更新显示时有多忙。能够做这些事情的软件被称为`并发软件`。

Java平台是从底层设计来支持并发编程的，在Java编程语言和Java类库中有基本的并发支持。从jdk5.0开始，Java平台还包含了高级并发api `java.util.concurrent`。
