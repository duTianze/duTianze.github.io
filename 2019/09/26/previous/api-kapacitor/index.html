定义任务时直接启用任务

POST /kapacitor/v1/tasks
&#123;
    "id" : "dutianze-t",
    "type" : "stream",
    "dbrps" : [&#123;"db": "DATABASE_NAME", "rp" : "RP_NAME"&#125;],
    "script" : "stream\n    |from()\n        .measurement('cpu')\n",
    "status" : "enabled"
&#125;

返回码

Code 说明
200 创建的任务, 并带有任务详情
404 任务不存在
